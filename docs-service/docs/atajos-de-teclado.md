# Atajos de Teclado

El visualizador incluye un sistema completo de atajos de teclado que permite navegar y controlar la aplicación de forma más eficiente.

## Acceso al Panel de Atajos

Se puede acceder al panel de atajos de teclado de las siguientes maneras:

1. Presionar la tecla `5` en el teclado
2. Presionar `?` para abrir la ayuda
3. Click en el ícono de teclado en la barra lateral izquierda

## Configuración

En el panel de atajos se puede:

- **Activar/Desactivar globalmente**: El toggle "Atajos de teclado activos" habilita o deshabilita todos los atajos a la vez.
- **Configurar individualmente**: Cada atajo puede ser habilitado o deshabilitado de forma independiente.
- **Restaurar valores**: El botón "Restaurar valores" devuelve todos los atajos a su configuración por defecto.

La configuración se guarda automáticamente en el navegador y persiste entre sesiones.

## Categorías y Atajos Disponibles

### Navegación

| Atajo | Acción |
|-------|--------|
| `1` | Abrir/cerrar panel de capas |
| `2` | Abrir/cerrar panel de polígonos |
| `3` | Abrir/cerrar panel de mapa base |
| `4` | Abrir/cerrar panel de herramientas |
| `5` | Abrir/cerrar panel de atajos de teclado |
| `Esc` | Cerrar panel actual |

### Dibujo de Polígonos

| Atajo | Acción |
|-------|--------|
| `D` | Iniciar modo de dibujo de polígonos |
| `E` | Activar modo de edición de polígonos |
| `X` | Activar modo de eliminación de polígonos |
| `Esc` | Cancelar dibujo o edición actual |

### Control de Capas

| Atajo | Acción |
|-------|--------|
| `Q` | Toggle visibilidad de la primera capa activa |
| `W` | Toggle visibilidad de la segunda capa activa |
| `R` | Toggle visibilidad de la tercera capa activa |
| `S` | Toggle todas las escalas de capas |

### Reproducción de Animaciones

| Atajo | Acción |
|-------|--------|
| `Espacio` | Play/Pause animación de la primera capa |
| `Shift + Espacio` | Play/Pause animación sincronizada |
| `←` | Frame anterior |
| `→` | Frame siguiente |
| `↑` | Aumentar velocidad de reproducción |
| `↓` | Reducir velocidad de reproducción |

### Herramientas del Mapa

| Atajo | Acción |
|-------|--------|
| `C` | Toggle panel de coordenadas |
| `B` | Toggle barra de escala |

### General

| Atajo | Acción |
|-------|--------|
| `?` | Mostrar ayuda (panel de atajos) |
| `+` | Acercar zoom |
| `-` | Alejar zoom |
| `0` | Restaurar vista inicial |

## Notas Importantes

- Los atajos **no se activan** mientras se escribe en campos de texto (inputs, textareas).
- Algunos atajos funcionan incluso cuando un campo está activo (como `Esc` para cancelar).
- Los tooltips de los botones muestran el atajo correspondiente cuando está disponible.
- Si deshabilitás un atajo específico, el tooltip ya no mostrará esa tecla.

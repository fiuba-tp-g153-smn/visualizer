# Uso General

## Interfaz

La interfaz de usuario está compuesta por los siguientes elementos:

- **Barra de Navegación**: Contiene el título de la aplicación y el menú de navegación.
- **Panel de Capas**: Contiene las capas meteorológicas y de referencia.
- **Mapa**: Contiene el mapa base y las capas operativas.

## Navegación

- **Zoom**: Rueda del mouse ó controles de la UI (+/-).
- **Desplazamiento**: Arrastrar mapa (Click izquierdo + Drag) o usar las flechas del teclado.

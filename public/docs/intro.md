# Welcome to Visualizator Documentation

This is the documentation for the Visualizator application.

## Key Features

- **Map Viewer**: Interactive map with various layers.
- **Layers**: Configure opacity, animation, and more.
- **Tiles**: Choose your preferred base map.

## Getting Started

Navigate through the application using the main menu on the left.

- Use the **Layers** icon to manage map layers.
- Use the **Map** icon to change the base map.

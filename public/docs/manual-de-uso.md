# Manual de Uso

Bienvenido a la guía de usuario de Visualizator. Esta aplicación le permite explorar mapas interactivos con capas meteorológicas avanzadas.

## Interfaz Principal

La interfaz está diseñada para ser intuitiva y fácil de usar.

![Visión General](docs/images/manual-overview.png)

### Componentes Principales

1.  **Menú Lateral**: Situado a la izquierda, le permite acceder a las diferentes herramientas como Capas y Configuración de Mapa.
2.  **Área de Mapa**: El visor central donde se presentan los datos geoespaciales interactivos.

## Gestión de Capas

Para visualizar datos meteorológicos, utilice el panel de capas.

![Menú de Capas](docs/images/manual-layers.png)

### Pasos para Activar una Capa

1.  Haga clic en el icono de **Capas** (Layers) en el menú lateral.
2.  Aparecerá el panel de configuración.
3.  Seleccione la casilla de la capa deseada (ej. **Radar**, **Temperatura**).
4.  La capa se cargará automáticamente en el mapa.

> [!TIP]
> Puede activar múltiples capas simultáneamente para comparar datos.

## Navegación

- **Zoom**: Utilice la rueda del ratón o los botones en pantalla.
- **Pan**: Haga clic y arrastre para mover el mapa.
